# 💰 AI Finance Assistant (Multi-Agent System)

An AI-powered web application that intelligently processes user queries using multiple specialized agents.

## 🚀 Features

- 🤖 Multi-agent system (Email, Finance, Insights)
- 🧠 Context-aware memory (chat history)
- 🎯 Task classification and routing
- 💬 Chat-based UI (Flask)
- ⚡ Real-time AI responses using OpenAI API

## 🛠️ Tech Stack

- Python (Flask)
- OpenAI API
- HTML/CSS (Chat UI)

## 📌 How it Works

1. User enters a query
2. System classifies the task
3. Routes to the appropriate AI agent
4. Generates a structured response
5. Maintains conversation memory
